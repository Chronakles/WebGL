---------- Instructions ----------

This project's goal was to learn the graphics pipeline through WebGL.
To create the scene the library gl_matrix was used.

---------- Instructions ----------

To run the program just open the index.html file.
You can change the axis on which the objects rotate.

---------- tested environment ----------

The code was executed on a Windows 10 machine with the Chrome Browser
(Version 89.0.4389.90 (Offizieller Build) (64-Bit) ).
The code was written on the exact same OS and development environment.

---------- additional and general remarks ----------

The fundation of the code was provided by the book 
"Edward Angel, Dave Shreiner - Interactive Computer Graphics. A Top-Down Approach with WebGL-Pearson" (2014).
